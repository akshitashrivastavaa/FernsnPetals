FNP Sales Analysis Dashboard

This project presents a comprehensive sales analysis dashboard designed to evaluate the performance of an online gifting business. It consolidates key metrics and visual insights to support data-driven decision-making across sales, marketing, and operations.

📊 Key Highlights
Tracks overall performance through total orders, revenue, and average customer spend
Analyzes delivery efficiency via order-to-delivery time
Breaks down revenue across occasions, product categories, and monthly trends
Identifies top-performing products and high-demand cities
Evaluates customer ordering patterns by time of day

🎯 Business Value

The dashboard enables to:

Identify high-revenue occasions and product categories
Understand seasonal and monthly sales trends
Optimize inventory and promotional strategies
Improve logistics by monitoring delivery timelines
Target high-performing geographic markets
🛠️ Tools & Approach

Developed using Excel-based data visualization techniques, including KPI cards, pivot charts, and interactive slicers for dynamic filtering.

💡 Summary

This dashboard delivers a clear, structured view of business performance, helping translate raw data into actionable insights for strategic growth.
